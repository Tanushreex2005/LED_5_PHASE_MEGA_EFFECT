# 💡 ESP32 LED Effects Show

A collection of 5 creative LED lighting effects 
programmed on ESP32 using 11 LEDs.

## ✨ Effects

| Effect | Duration | Description |
|--------|----------|-------------|
| Chase Effect | ~1.1s | LEDs light up one by one |
| Build Line Effect | ~15s | LEDs build from both directions |
| Mega Effect | ~25s | Comet, pulse, twinkle, strobe combo |
| Center Burst Effect | ~4s | LEDs radiate outward from center |
| Binary Counter Effect | ~8s | LEDs count in binary |

**Total cycle: ~60 seconds**

## 🔧 Hardware Used

| Component | Details |
|-----------|---------|
| ESP32 Dev Module | Main microcontroller |
| LEDs | 11 LEDs |
| Resistors | 220Ω (one per LED) |
| Breadboard | For connections |

## 📌 Pin Connections
GPIO 2, 4, 5, 13, 15, 18,
19, 21, 22, 23, 26
